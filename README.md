一个体验更加好的树洞客户端。

### 已实现的功能

- 浏览树洞
- 搜索关键词或 PID

### 用法

修改 config.json 中的 "token" 项为你的 token（在树洞网页版中点击账户 -> 复制 User Token），然后运行 PKU-Hole.exe 即可。
